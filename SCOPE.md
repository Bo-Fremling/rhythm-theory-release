# Scope

This is a **FULL** release.

Included:
- `00_TOP/` (Core + locks + tooling)
- `00_TOP/OVERLAY/` (Compare reference data; used only by Compare)
- `verify/` (verification scripts)
- Reviewer artifacts generated by `sm29_report.py`:
  - `00_TOP/LOCKS/SM_PARAM_INDEX/SM_29_REPORT.md`
  - `out/SM29_PAGES.md`

Policy:
- **Core must never** use overlay/refs for selection/optimization.
- **Compare may** read overlay/refs but must not feed back into Core.
